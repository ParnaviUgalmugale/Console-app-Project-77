What is visual studio code and how to use it ?

Visual studio code is a coding app where we can create console and javascripts. It has more efficient features than other apps and is easy to use. Once Visual studio code is downloaded, we must install the live server using extensions, which helps in seeing the outputs of our code. Next, we must go to explorer, where we should create html, css and js files where we are supposed to code. once code is done, we can click on go live on the bottom right corner and see the output. coding on this app is really easy. Once you code more than 3 times, you'll get used to the app.


How to open the console screen ?

Right click on any page or website. Click on inspect. Once another pop up appears, go to console on the top beside elements. This is how you can open the console to any screen.
